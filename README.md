# St.Branco | Portfolio

Um portfólio pessoal moderno e responsivo, construído com HTML, CSS e JavaScript.

## 🎨 Características

- **Dark Mode** com estrelas animadas
- **Responsive Design** otimizado para mobile e desktop
- **Animações Suaves** em transições e elementos
- **Navegação Sticky** com efeitos hover
- **Skills Section** com grid interativo
- **Contacto Funcional** com mailto integrado

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilos e animações
- **JavaScript** - Interatividade (dark mode, estrelas animadas)
- **Google Fonts** - Tipografia profissional (Caudex, Croissant One)

## 📱 Tema Dia/Noite

- **Modo Light**: Gradiente azul claro, profissional e limpo
- **Modo Dark**: Azul marinho profundo com estrelas animadas que piscam

## 💡 Funcionalidades

### Dark Mode Toggle
Clica no botão 🌙 para alternar entre modo claro e escuro. As transições são suaves e profissionais.

### Animações
- Hero descende suavemente ao carregar
- Imagem flutua continuamente
- Cards de skills sobem ao hover
- Botão de contacto escala ao hover

### Navegação
Menu sticky com underline animado nos links. Navegação fluida e intuitiva.

## 📁 Estrutura do Projeto## 🔧 Como Usar

1. Clone o repositório
2. Abra `index.html` no browser
3. Explore o dark mode e as animações
